# Portfolio Website Design

## Visual Design

### Color Scheme
- Primary: Purple gradient (#667eea to #764ba2)
- Background: Light gray (#f5f5f5)
- Text: Dark gray (#333)
- Accent: Purple (#667eea)

### Sections

1. **Hero Section**
   - Full-width purple gradient background
   - Name in large, bold text
   - Professional title
   - Contact information (email, phone, location)

2. **Navigation Bar**
   - Sticky navigation
   - Links to all sections: Profile, Skills, Experience, Education, Leisure
   - Smooth scroll behavior

3. **Personal Profile**
   - Clean white card
   - Professional summary
   - Easy to read paragraphs

4. **Skills & Tools**
   - Two-column grid layout
   - Core skills on left
   - Tools & technologies on right
   - Bullet points with custom icons

5. **Work Experience**
   - Vertical timeline with gradient line
   - Each job displayed as a timeline item
   - Company, position, dates, location
   - Description and key highlights
   - Technology stack badges

6. **Education**
   - Clean card layout
   - University degree and secondary education
   - Grades and dates clearly displayed

7. **Leisure Activities**
   - About your personal interests
   - Running achievements
   - Tech community involvement
   - Personal development projects

8. **Footer**
   - Dark background
   - Copyright information

## Responsive Design

- Mobile-friendly
- Adjusts layout for tablets and phones
- Readable on all screen sizes

## Typography

- Font: Segoe UI (professional, modern)
- Headings: Bold and clear
- Body text: Easy to read with good line spacing

## Interactive Elements

- Smooth scrolling navigation
- Hover effects on links
- Timeline markers
- Professional color transitions
